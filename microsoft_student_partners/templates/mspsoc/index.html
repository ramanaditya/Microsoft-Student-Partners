We are a bunch of MSP (Microsoft Student Partners) who came up with "MSP Summer of Code". The aim behind this initiative is to mentor you on the Open Source Projects, particularly to get you accustomed with Open Source. Apart from mentorship, we would provide you with potential exclusive certificates, goodies and vouchers on successful completion of this 60 Days long program.

FAQs

Do I need to pay any money to register for the Event ?
No. You do not have to pay anything to anyone to register yourself for this Sumemr of Code.
Do I need to have any specific qualifications to be a participant for the Summer of Code ?
If you love to code and innovate, you are more than welcome to participate in the Event.
How do I submit my project for this event ?
You have to develop the application and put down all the codes on GitHub and from there only we will track your submissions and contributions.
Do we need to have the entire idea fully working ?
The entire idea need not be fully implemented however, the submission should be functional so that it can be reviewed.
Does one have to be online and available for the entire duration of the Event ?
No, one does not need to be online for the entire duration. You can develop the project on your local system and push the changes on GitHub repository of the respective project.
Can Microsoft employees participate ?
All employees currently employed by Microsoft can participate in this hackathon.
Who will own the IP(Intellectual Property) Rights to the product that I have built ?
The developer/developers of the web/mobile application will have all rights and own the IP of the product. However, all code needs to be in public domain (open source) so that it can be reviewed.
